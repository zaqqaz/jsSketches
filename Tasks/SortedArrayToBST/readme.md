#Sorted Array to Balanced BST
Given a sorted array. Write a function that creates a Balanced Binary Search Tree using array elements.

Examples:
```
Input:  Array {1, 2, 3}
Output: A Balanced BST
     2
   /  \
  1    3 

```
```

Input: Array {1, 2, 3, 4}
Output: A Balanced BST
      3
    /  \
   2    4
 /
1
```
